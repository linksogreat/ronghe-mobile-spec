# 融和移动端 UI 视觉规范系统

这是一个基于 Vant 4 的移动端 UI 视觉规范展示与管理系统。不仅提供了静态的规范展示，还集成了 Node.js 后端，支持在线编辑和保存设计规范（色彩、字体、组件规约等）。

## 功能特性

- **完整规范展示**：包含色彩、字体、图标、布局及几十种常用组件的详细设计规约。
- **动态演示**：集成了 Vant 4 组件的交互演示，右侧配备手机模拟器实时预览。
- **在线编辑模式**：
  - 管理员登录（密钥认证）。
  - 可视化编辑色彩、字号、间距等基础样式。
  - 在线修改组件的设计规约文本。
  - 在线编辑组件的代码示例（支持 Vue Template 和 HTML）。
  - 数据持久化保存到服务端 JSON 文件。

## 快速开始

### 1. 环境准备
确保您的电脑已安装 [Node.js](https://nodejs.org/) (建议 v14.0 以上)。

### 2. 安装依赖
在项目根目录下打开终端，运行：
```bash
npm install
```

### 3. 启动服务
```bash
npm start
```
或者直接运行：
```bash
node server.js
```

### 4. 访问项目
服务启动后，打开浏览器访问：
http://localhost:3000/index.html

## 使用指南

### 进入编辑模式
1. 在页面左上角，点击标题栏右侧的 **锁形图标**。
2. 在弹出的登录框中输入密钥：`Rong@123`。
3. 验证通过后，页面将进入编辑模式，锁图标变为解锁状态。

### 编辑样式 (1. 样式规范)
- 进入“样式规范”页面。
- 可以直接修改色板的名称、HEX 值、描述。
- 可以修改字体的用途、字号、颜色等。
- 修改完成后，点击右上角的 **“保存修改”** 按钮。

### 编辑组件 (3. 常规 / 4. 导航等)
- 选择任意组件。
- **设计规约**：点击“添加”按钮新增规约，或直接修改现有规约文本。
- **代码示例**：可以直接在代码框中编辑 HTML 或 Vue Template 代码。
- **示例标题**：可以修改示例的标题。
- 修改完成后，点击右上角的 **“保存修改”** 按钮。

### 重置数据
如果修改有误，可以点击右上角的 **“重置”** 按钮，放弃当前未保存的修改，重新加载服务器数据。

## 项目结构

```
/
├── assets/              # 静态资源 (CSS, JS, Fonts)
├── data/                # 数据存储 (JSON)
│   ├── data.json        # 样式规范数据
│   └── components.json  # 组件规范数据
├── pages/               # 前端页面
│   ├── 1_style.html     # 样式规范页
│   ├── 3_components.html# 组件详情页
│   └── mobile_preview.html # 手机预览页
├── server.js            # Node.js 服务端入口
├── package.json         # 项目依赖配置
└── 融和移动端规范.html    # 主入口页面
```

## 技术栈
- **Frontend**: HTML5, Tailwind CSS, Vue 3, Vant 4
- **Backend**: Node.js, Express
- **Storage**: JSON File System
